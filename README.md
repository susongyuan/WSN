# WSN - 无线传感器网络覆盖调度优化

本项目针对异构无线传感器网络中的 **k-coverage 最大寿命调度问题**，提出了分段变能量遗传算法（PVEGA），并与 STHGA、MCASS 算法进行了全面的对比实验和消融实验。

## 研究贡献

### 算法创新
- 提出**分段自适应能量遗传算法（PVEGA）**，融合分段优化策略分解 k-coverage 问题
- 结合**快速冗余调度机制**动态筛选传感器
- 设计**同源传感器标准化方案**统一处理异质激活时长

### 优化策略
- 创新激活时间标准化方法，将非均匀激活传感器拆分为多同源单元
- 实现异构网络等效转换，大幅降低计算复杂度

### 效能突破
- 网络寿命提升 **15.23%-20.85%**（p<0.01）
- 覆盖冗余降低 **32%-67%**
- 计算效率较 STHGA 算法提升 **1-3 个数量级**

## 三种算法对比

### 1. PVEGA（本文提出）
- **核心思想**：能量标准化 + 分段遗传优化 + 快速冗余调度
- **特点**：处理异构能量，支持 OpenMP 并行加速
- **参数**：种群20，交叉率0.8，变异率0.05，每阶段50代

### 2. STHGA（基线算法）
- **核心思想**：混合遗传算法 + 前向/混合/关键调度
- **参考**：IEEE TEVC 2010
- **参数**：种群3，K1=K2=5，变异间隔20代

### 3. MCASS（贪心对比）
- **核心思想**：多阶段覆盖集贪心调度
- **特点**：速度最快，但网络寿命优化效果较差

## 输入数据格式

每行三个数值：`x坐标 y坐标 能量值`

```
10.088384829722886 5.005613690037025 16
21.641553223245314 5.756543008081688 15
42.26782403190827 7.543218679033103 12
```

- 部署区域：50×50
- 传感器规模：500 ~ 10,000
- 能量范围：1 ~ 20

## 输出格式

覆盖子集方案（`output_N_R.txt`）：
```
1, (s_7_4, s_10_1, s_13_10, ..., s_492_16)
2, (s_4_3, s_15_5, s_28_3, ..., s_496_9)
```
其中 `s_X_Y` 表示第 X 个传感器的第 Y 个子单元。

## 编译与运行

```bash
# PVEGA（推荐使用OpenMP加速）
g++ -std=c++11 -fopenmp PVEGA对比实验/PVEGA对比实验.cpp -o pvega
./pvega

# MCASS
g++ -std=c++11 MCASS对比实验/MCASS对比实验.cpp -o mcass
./mcass

# STHGA
g++ -std=c++11 sthaga对比实验/sthga对比试验.cpp -o sthga
./sthga
```

运行前确保对应目录下有 `input_*.txt` 文件。
